## De-concierge Non-Negotiable Build Guides

- Beautiful, polished UI at every stage.
- User-focused decisions, from onboarding to GTM strategy.
- Story-first explanations: each tool or component maps to a one-sentence pain-point solution.
- ENS-powered identity baked into the MVP for host trust, guest recognition, and explainable interactions.
- Verifiable booking ledger: every record must be auditable end-to-end (Polygon + Filecoin Onchain Cloud + Attestations).
