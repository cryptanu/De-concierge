# De-concierge Demo Script (Draft)

1. **Hook (Problem & Story)**
   - “In the last year alone we double-booked the same loft three times…”
   - Show hero UI: intent prompt, ENS-branded host identity, Filecoin-backed proof badge.

2. **Walkthrough**
   - Guest enters natural-language intent.
   - AI concierge returns curated listings with “why this fits” tags.
   - Booking flow: wallet signs permit, Polygon tx hash + ENS host shown.
   - Agent confirms availability (explain attestation + duplicate check with LLM reasoning snippet).
   - Host dashboard: conflict alert resolved, verifiable timeline, export ICS.

3. **Tech Highlights for Judges**
   - Polygon contracts (agentic payments + USDC logic).
   - ENS subnames & text records fueling UI identity.
   - Flare FDC attestation evidence.
   - Filecoin Onchain Cloud for raw data + photo assets.
   - The Graph index powering analytics & AI.

4. **Prize Callouts**
   - Polygon, ENS, Circle, The Graph, Flare, Filecoin — one sentence each connecting to their criteria.

5. **Closing**
   - “De-concierge replaces calendar whack-a-mole with a verifiable ledger your guests, hosts, and platforms can all trust.”
   - CTA: roadmap for platform integrations + go-to-market.
